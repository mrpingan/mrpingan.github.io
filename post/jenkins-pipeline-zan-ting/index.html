jenkins pipeline暂停

2020-07-01

pipeline中可以进行暂停操作，这在进行高风险操作的时候特别有用。比如，有两台服务器，部署了一台，需要先测试观察日志，就可以在部署这台的stage后加入暂停stage。

stage('input') {
		input {
				message "Should we continue?"
				ok "Yes, we should."
		}
		steps {
				echo "continue..."
		}
}

当加入这段groovy脚本后，jenkins执行到这儿就会需要用户手动去点击"Yes, we should"。这样，脚本才会继续跑下去。

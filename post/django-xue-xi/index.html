命名空间

当url有多个组时候，为了避免url重复导致请求被路由到其他视图上，可以使用命名空间的方式，这样即使不通命名空间的url一样，也不会影响。
1、在子url中指明app_name


from django.urls import path
from . import views

app_name = 'blog'

urlpatterns = [
    path('',view.post),
]

2、在父url中引用namespace


from django.urls import path,include
from django.contrib import admin

urlpatterns = [
    path('blog/',include('blog.urls',namespace='blog')),
]

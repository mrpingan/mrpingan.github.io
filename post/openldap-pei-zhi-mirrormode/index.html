openldap 配置mirrormode

2019-11-08

简介

MirrorMode 模式镜像同步模式，而且主服务器互相以推的方式实现目录树条目同步，最多允许且两台机器为主服务器。如果要添加更多节点，此时只能添加多台从服务器，而不能将添加的节点配置为主服务器

准备

用途 IP 备注
provider1 192.168.33.10
provider2 192.168.33.11

两台服务器互为主备，但是只能在一个provider上写。

安装ldap

apt install slapd ldap-utils

#重新配置
dpkg-reconfigure slapd

dn: dc=ping,dc=cn
管理员密码：admin

两台服务器的DIT要保持一致

开启日志(如果已开启可忽略)

log.ldif文件内容如下：

dn: cn=config
changetype: modify
replace: olcLogLevel
olcLogLevel: stats sync

应用配置：

ldapmodify -Y EXTERNAL -H ldapi:/// -f log.ldif

加载syncprov模块

sync.ldif文件如下：

dn: cn=module{0},cn=config
changetype: modify
add: olcModuleLoad
olcModuleLoad: syncprov.la

dn: olcOverlay=syncprov,olcDatabase={1}hdb,cn=config
changetype: add
objectClass: olcOverlayConfig
objectClass: olcSyncProvConfig
olcOverlay: syncprov
olcSpCheckpoint: 100 10
olcSpSessionLog: 100

注意：

  • 指定的dn存在
  • 指定的Database要存在
  • 模块文件存在，ubuntu 的模块目录一般在/usr/lib/ldap目录下,如果在其他目录下，可通过olcModulePath指定目录，也可查看cn=module{0}.ldif文件olcModulePath是否被指定

应用配置：

ldapmodify -Y EXTERNAL -H ldapi:/// -f sync.ldif

配置mirror模式

master01.ldif文件内容如下：
master01的ip地址为: 192.168.33.10

dn: cn=config
changetype: modify
replace: olcServerID
olcServerID: 1

dn: olcDatabase={1}hdb,cn=config
changetype: modify
replace: olcSyncRepl
olcSyncRepl: rid=001
             provider=ldap://192.168.33.11:389
             bindmethod=simple
             binddn="cn=admin,dc=ping,dc=cn"
             credentials=admin
             searchbase="dc=ping,dc=cn"
             filter="(objectClass=*)"
             scope=sub
             schemachecking=off
             attrs="*,+"
             type=refreshAndPersist
             retry="5 5 300 +"
             interval=00:00:01:00
-
add: olcMirrorMode
olcMirrorMode: TRUE
-
add: olcDbIndex
olcDbIndex: entryUUID eq
-
add: olcDbIndex
olcDbIndex: entryCSN eq

注意：

  • 此处的provider指定的另一台服务器的ip
  • rid要保持一致
  • olcServerID要区分

应用配置：

ldapmodify -Y EXTERNAL -H ldapi:/// -f master01.ldif

master02.ldif文件内容如下：

dn: cn=config
changetype: modify
replace: olcServerID
olcServerID: 2

dn: olcDatabase={1}hdb,cn=config
changetype: modify
replace: olcSyncRepl
olcSyncRepl: rid=001
             provider=ldap://192.168.33.10:389
             bindmethod=simple
             binddn="cn=admin,dc=ping,dc=cn"
             credentials=admin
             searchbase="dc=ping,dc=cn"
             filter="(objectClass=*)"
             scope=sub
             schemachecking=off
             attrs="*,+"
             type=refreshAndPersist
             retry="5 5 300 +"
             interval=00:00:01:00
-
add: olcMirrorMode
olcMirrorMode: TRUE
-
add: olcDbIndex
olcDbIndex: entryUUID eq
-
add: olcDbIndex
olcDbIndex: entryCSN eq

应用配置:

ldapmodify -Y EXTERNAL -H ldapi:/// -f master02.ldif
